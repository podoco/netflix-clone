### `기능` 
 
* Axios를 활용하여 movie db api 요청
* styled component 사용하여 꾸미기
* 영화를 상세히 보여주는 모달 생성
* useParams 이용하여 영화 상세페이지 구현하기
* useLocation을 이용한 검색 페이지 구현하기
* 모달 창 외부 클릭시 모달 닫게 만드는 custom hooks
* swiper 모듈을 이용한 터치 슬라이드 구현
